# Real-Time Object Detection using YOLOv5 and SSD

## 🔹 Project Overview
This project implements and compares YOLOv5 and SSD architectures for real-time object detection tasks.

## 🔹 Objectives
- Perform real-time object detection
- Compare YOLOv5 and SSD performance
- Analyze speed vs accuracy trade-off

## 🔹 Model Architectures
- YOLOv5 (one-stage detector)
- SSD (Single Shot MultiBox Detector)

## 🔹 Training Pipeline
1. Data preprocessing and annotation
2. Model training
3. Hyperparameter tuning
4. Evaluation

## 🔹 Evaluation Metrics
- mAP (mean Average Precision)
- IoU (Intersection over Union)
- Precision-Recall Curve
- Inference Speed

## 🔹 Technologies Used
Python, PyTorch / TensorFlow, OpenCV, NumPy, Matplotlib
